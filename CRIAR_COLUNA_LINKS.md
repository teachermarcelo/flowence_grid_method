# 🔗 Criar Coluna Links no Supabase

Execute este comando **exatamente** no SQL Editor do Supabase:

```sql
ALTER TABLE flowence_class ADD COLUMN links TEXT;
```

**Clique em RUN (Ctrl+Enter)**

---

## ✅ Se der erro "column already exists"

Ignore, significa que a coluna já existe.

---

## ✨ Pronto!

Depois que rodar o SQL, a coluna `links` estará criada e você pode:
1. Recarregar a página
2. Clicar em "+ Nova Turma"
3. Rolar até "Links Externos"
4. Adicionar quantos links quiser
